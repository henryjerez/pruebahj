LECTURAS DE INTERES

El presente OVI sobre el curso de Diseño de Sitios Web está dirigido a personas interesadas en el diseño y administración de sitios web.

Permite la profundización sobre el uso de GIT y GIHUB para gestionar proyectos colaborativos de diseño web, usando lenguaje HTML5 y CSS3 para creación de contenidos en formato web.

El curso esta conformado por: Introducción al Diseño Web.
